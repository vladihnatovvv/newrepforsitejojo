# Підключення WordPress + WooCommerce

Цей файл описує короткий порядок запуску магазину на вже підготовленій темі `jojo-avoe`.

## 1. Підготувати WordPress

Потрібно:

- встановити `WordPress`
- скопіювати тему `wp-content/themes/jojo-avoe`
- активувати тему в `Appearance -> Themes`

## 2. Встановити WooCommerce

У WordPress:

- `Plugins -> Add New`
- знайти `WooCommerce`
- встановити
- активувати

Після активації:

- пройти стандартний wizard
- створити сторінки `Shop`, `Cart`, `Checkout`, `My account`

## 3. Поставити тему магазину

Після активації теми:

- призначити `Home` як головну
- підключити меню
- перевірити логотип, телефон, email у кастомайзері

## 4. Завести категорії

Є готовий файл:

- `data/woocommerce-categories.csv`

Його можна використати як опорний список для ручного створення категорій або для кастомного імпорту.

Рекомендована структура:

- Головні убори
  - Кепі
  - Кепі-реглан
  - Морячка
  - Кепки
  - Солома
  - Кепка-докерка
  - Панами
- Аксесуари
  - Футляри
  - Хустки
  - Окуляри
- Сумки
- SALE

## 5. Імпорт товарів

Є приклад файлу:

- `data/sample-products.csv`

Його треба використовувати як шаблон під ваш реальний імпорт.

Мінімальні поля:

- `SKU`
- `Name`
- `Regular price`
- `Categories`
- `Images`
- атрибути кольору / розміру, якщо є

У WordPress:

- `WooCommerce -> Products -> Import`
- завантажити CSV
- перевірити мапінг полів

## 6. Що підключити далі

Після базового запуску:

- `Нова пошта`
- `keyCRM`
- `Torgsoft`
- SEO-плагін
- кеш
- SMTP

## 7. Що обов'язково перевірити після запуску

- головна сторінка
- каталог
- товар
- кошик
- checkout
- мобільна версія
- додавання товару в кошик
- відправка тестового замовлення
- коректність категорій
- роботу цін, SKU і фото
